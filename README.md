# MiniMax Router

> MiniMax 多模态路由技能 — 图片 / 视频 / 音乐 / 语音 / 对话

## 功能概览

| 能力 | 模型 | 每日限额 |
|------|------|----------|
| 🖼️ 图片生成 | `image-01` | 120 张/天 |
| 🎬 视频生成 | `MiniMax-Hailuo-2.3` / `Fast` | 2 个/天 |
| 🎵 作曲 | `music-2.5` | 4 首/天 |
| 🔊 语音合成 | `speech-2.8-hd` | 11,000 字/天 |
| 💬 文本对话 | `MiniMax-M2.7` | 无限制 |

---

## 使用方式

### 自然语言（推荐）

系统自动识别意图，直接说就行：

```
"帮我生成一张海边日落图"
"做个日出视频"
"写首摇滚歌曲"
"把这段文字转语音"
```

### 斜杠命令

```
/c <内容>   →  文本对话
/t <内容>   →  文本转语音
/v <内容>   →  文本/图片转视频
/m <内容>   →  作曲
/i <内容>   →  文本转图像
```

---

## 交互流程

### 🖼️ 生成图片

需要提供：**内容描述 + 比例**

比例选项：①1:1 ②16:9 ③9:16 ④4:3 ⑤3:4

```
用户：帮我生成图片，海边日落，16:9
→ ⚠️ 图片生成已准备好 | 内容：海边日落 | 比例：16:9
   回复「生成」确认
用户：好
→ 执行
```

> 参数齐全时直接执行；参数不全时追问。

---

### 🎬 生成视频

提供描述即可，默认 768P / 6秒 / Fast模型

```
用户：帮我做视频，海边日落延时摄影
→ 执行（文字生视频用 2.3 模型）
```

**运镜指令（可选）：**
`[左移] [右移] [左摇] [右摇] [推进] [拉远] [上升] [下降] [上摇] [下摇] [变焦推近] [变焦拉远] [晃动] [跟随] [固定]`

---

### 🎵 作曲

**第一步：选类型**
```
用户：我要作曲
我：请选择类型：①纯音乐（无人声）②有人声歌曲
```

**第二步：仅有人声歌曲时追问歌词**
```
用户：① → 直接执行纯音乐
用户：② → 请提供歌词内容（回歌词或"无"自动生成）
```

---

### 🔊 语音合成

需要提供：**音色 + 文字内容**

音色选项：
| 序号 | 名称 | 风格 |
|------|------|------|
| ① | 温润青年 | Chinese (Mandarin)_Gentle_Youth |
| ② | 沉稳高管 | Chinese (Mandarin)_Reliable_Executive |
| ③ | 电台男主播 | Chinese (Mandarin)_Radio_Host |
| ④ | 新闻女声 | Chinese (Mandarin)_News_Anchor |
| ⑤ | 甜美女声 | Chinese (Mandarin)_Sweet_Lady |
| ⑥ | 温暖少女 | Chinese (Mandarin)_Warm_Girl |

```
用户：帮我转语音
我：请选择音色（①-⑥）并提供文字内容
用户：③，海边的风吹过来很舒服
→ 执行
```

---

## ⚠️ 调用规范

**有配额的 API（图片/视频/音乐/语音合成）必须遵守：**

1. 调用一次 → 等待结果返回
2. 结果到手 → 展示给用户
3. 用户确认 → 才决定是否继续

> 一次只调一个，等结果再说话。禁止连续调用耗尽配额。

---

## 模型选择规则

| 场景 | 模型 | 说明 |
|------|------|------|
| 纯文字生成视频 | `MiniMax-Hailuo-2.3` | 默认 |
| 提供图片生成视频 | `MiniMax-Hailuo-2.3-Fast` | 更快 |
| ⚠️ 有图用 Fast | — | 不能纯文字用 Fast |

## 配置

环境变量：
```
MINIMAX_API_KEY=<your-api-key>
```

API Key 位置：`~/.openclaw/openclaw.json` → `providers.minimax.apiKey`
