# 视频运镜指令速查

> 适用模型：`MiniMax-Hailuo-2.3`（文字生视频）、`MiniMax-Hailuo-2.3-Fast`（图生视频）
>
> 官方文档：<https://platform.minimaxi.com/docs/api-reference/video-generation>

## 用法

在视频生成的 `prompt` 字段里用方括号包住运镜关键词，控制镜头运动：

```
[左移] 一只猫在窗台上看日落，电影感，4K
[推进] 特写主角脸部表情，戏剧光
```

运镜可以叠加（按顺序执行）：

```
[推进][上摇] 镜头从脚部推到脸部再向上摇
```

## 完整指令列表

| 指令 | 效果 |
|------|------|
| `[左移]` | 镜头水平向左移动（Pan Left） |
| `[右移]` | 镜头水平向右移动（Pan Right） |
| `[左摇]` | 摄像机向左摇（围绕主体左转） |
| `[右摇]` | 摄像机向右摇（围绕主体右转） |
| `[推进]` | 镜头向前推进（Zoom In / Dolly In） |
| `[拉远]` | 镜头向后拉远（Zoom Out / Dolly Out） |
| `[上升]` | 镜头垂直上升（Crane Up） |
| `[下降]` | 镜头垂直下降（Crane Down） |
| `[上摇]` | 镜头向上摇（Tilt Up） |
| `[下摇]` | 镜头向下摇（Tilt Down） |
| `[变焦推近]` | 焦距拉近（光学变焦，画面压缩感） |
| `[变焦拉远]` | 焦距拉远（广角，画面拉伸感） |
| `[晃动]` | 摄像机抖动（手持感，紧张/运动场景） |
| `[跟随]` | 镜头跟随主体移动（Tracking） |
| `[固定]` | 镜头完全固定（静态画面） |

## 实战 Prompt 模板

### 风光 / 延时

```
[推进] 黄昏时分的海边小镇延时摄影，
[上升] 镜头从街道缓缓升起，越过屋顶看到远处大海
```

### 人物 / 短剧

```
[推进] 一个中年男人坐在书桌前沉思，
[左摇] 镜头慢慢左转揭示身后的书架，
[上摇] 镜头向上摇到吊灯
```

### 产品 / 商业

```
[拉远] 一瓶香水从近景特写慢慢拉远到全景，
[跟随] 镜头跟着香水瓶在桌面上滑动
```

### 动作 / 紧张

```
[晃动][跟随] 一个人在雨夜的街道奔跑，
手提摄像机抖动跟拍，电影感
```

## 限制

- 默认视频时长 **6 秒**、分辨率 **768P**
- 运镜指令放 `prompt` 字段里，**用方括号**而不是圆括号
- 不支持组合长 prompt 里的运镜词（如 `[电影感的镜头移动]`）——只能从上面 15 个关键词里选
