# MiniMax Router

## 核心功能

### 1. 自然语言自动路由（无需命令）

系统自动识别意图并调用对应模型：

| 意图 | 触发关键词 | 示例 |
|------|-----------|------|
| 图片 | 生成图片、画一张、做个图 | "帮我生成一张海边日落图" |
| 视频 | 生成视频、做个视频 | "帮我做一个日出视频" |
| 视频+配音+音乐 | 视频配音、视频旁白、视频音乐 | "生成视频并配音" |
| 音乐 | 写首歌、作曲 | "帮我写首摇滚歌曲" |
| 语音 | 转语音、读出来、配音 | "把这段文字转语音" |
| 对话 | 其他所有 | "今天天气怎么样" |

### 2. 显式斜杠命令

```
/c <内容>   →  文本对话
/t <内容>   →  文本转语音
/v <内容>   →  文本/图片转视频
/m <内容>   →  作曲
/i <内容>   →  文本转图像
```

### 3. 链式组合（自然语言）

用连接词组合多个操作：

```
生成一张海边日落图，然后转语音
        ↓
1. 生成图片 → 2. 把描述转语音
```

### 4. 视频+配音+音乐

一句话搞定完整视频：

```
"生成视频：海边日落，旁白：今天天气真好，配乐：轻快钢琴曲"
        ↓
1. 生成视频
2. 生成 TTS 配音
3. 生成背景音乐
4. FFmpeg 合并为最终视频
```

## ⚠️ 调用规范（必须遵守）

### 引导交互流程

图片、视频、音乐、语音合成，参数不全时必须追问清楚，确认后再执行。参数齐全时直接执行。

---

#### 🎵 作曲

**第一步：问类型**
```
用户：我要作曲
我：请选择类型：
    ① 纯音乐（无人声）
    ② 有人声歌曲
```

**第二步：仅有人声歌曲时追问歌词**
```
用户：①
→ 直接执行纯音乐（不问歌词）

用户：②
→ 请提供歌词内容（直接回歌词，或回"无"自动生成）
用户：我的歌词是...
→ 执行
```

---

#### 🖼️ 生成图片

```
用户：我要生成图片
我：请一次性告诉我：
    1. 图片内容/主题
    2. 比例：①1:1 ②16:9 ③9:16 ④4:3 ⑤3:4

用户：海边日落，②
→ 执行
```

> 参数齐全时（如"帮我画一张海边日落图"）直接执行，不追问。

---

#### 🎬 生成视频

```
用户：帮我做视频（或直接发来一张图片）
→ 直接执行，默认 768P / 6秒 / Fast模型
```

> 有图时自动用 Fast 模型；无图时用 2.3 模型。
> 分辨率默认 768P，时长默认 6秒，无需询问。
> 视频描述支持运镜指令：`[左移] [右移] [左摇] [右摇] [推进] [拉远] [上升] [下降] [上摇] [下摇] [变焦推近] [变焦拉远] [晃动] [跟随] [固定]`

---

#### 🔊 语音合成

```
用户：我要转语音
我：请一次性告诉我：
    1. 音色：①温润青年 ②沉稳高管 ③电台男主播 ④新闻女声 ⑤甜美女声 ⑥温暖少女
    2. 要转换的文字内容

用户：③，海边的风吹过来很舒服
→ 执行
```

---

### 确认机制

参数齐全时，执行前仍需确认：
```
用户：帮我生成一张海边日落图，16:9
→ ⚠️ 图片生成已准备好。
  内容：海边日落 | 比例：16:9
  回复「生成」确认执行，其他内容取消。
用户：生成
→ 执行
```

### 普通对话
文字对话直接执行，无需确认。

## 使用示例

### 自然语言 + 引导交互

```
用户：我要作曲
我：请选择类型：①纯音乐 ②有人声歌曲
用户：①
→ 执行纯音乐

用户：我要作曲
我：请选择类型：①纯音乐 ②有人声歌曲
用户：②
我：请提供歌词（回歌词或"无"自动生成）
用户：无
→ 执行

用户：帮我画一张海边日落图
→ ⚠️ 图片生成已准备好，回复「生成」确认

用户：帮我生成图片，海边日落，16:9
→ ⚠️ 图片生成已准备好 | 内容：海边日落 | 比例：16:9
   回复「生成」确认
用户：好
→ 执行
```

### 斜杠命令

```
/i 图片描述       → 生成图片
/v 视频描述       → 生成视频
/t 内容           → 语音合成
/c 内容 /t        → 对话后转语音
/i 描述 /t        → 生成图片后转语音
```

## API 配置

环境变量：
- `MINIMAX_API_KEY` — MiniMax Token Plan API 密钥

## 视频模型选择（自动判断）

| 你提供的内容 | 调用的模型 | 说明 |
|-------------|-----------|------|
| **只有文字描述** | `MiniMax-Hailuo-2.3` | 文字生成视频，直接说就行 |
| **提供了图片** | `MiniMax-Hailuo-2.3-Fast` | 图生视频，速度更快 |

**规则：**
- 有图 → Fast
- 无图 → 2.3
- ⚠️ Fast 模型**不支持**纯文字生成视频

## 模型对应

| 功能 | 模型 | Token Plan 限额 (Max) |
|------|------|----------------------|
| 语音 TTS | speech-2.8-hd | 11,000 字/天 |
| 图片 Image | image-01 | 120 张/天 |
| 视频 Video | MiniMax-Hailuo-2.3 / Fast | 2 个/天 |
| 音乐 Music | music-2.5 | 4 首/天 |

## 脚本

- `router.py` — 自动路由主脚本
- `tts.py` — 语音合成
- `image.py` — 图像生成
- `video.py` — 视频生成
- `music.py` — 音乐生成
- `video_with_audio.py` — 视频+配音+音乐合并
